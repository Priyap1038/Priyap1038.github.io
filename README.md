# 🌐 Priya Patole — Personal Portfolio

A modern, fully responsive personal portfolio website built with **React + Vite**, featuring a premium dark/light theme, glassmorphism UI, animated components, and a showcase of skills, projects, and contact information.

---

## ✨ Features

- 🎨 **Dark & Light Mode** — Pill-style toggle with animated sun/moon switch
- 💎 **Glassmorphism Design** — Frosted-glass cards with violet/cyan gradient accents
- ⚡ **Animated Hero** — Floating background blobs, typing effect, gradient heading
- 🖼️ **Project Carousel** — Image slider for each project with dot navigation
- 📊 **GitHub Stats** — Live streak stats, top languages, and contribution calendar
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile
- 🚀 **Vite Powered** — Lightning-fast dev server and build

---

## 🗂️ Sections

| Section | Description |
|---|---|
| **Home** | Animated hero with name, typed roles, and CTA buttons |
| **About** | Bio, floating profile photo, stats grid, resume download |
| **Skills** | Glassmorphism cards for all technologies and tools |
| **Projects** | Cards with image carousels, tech stack, GitHub & live links |
| **GitHub Stats** | Live GitHub streak, top languages, stats card, and calendar |
| **Contact** | Contact info + message form with neon-focus inputs |

---

## 🛠️ Tech Stack

- **Framework**: React 18 with Vite
- **Styling**: Vanilla CSS (CSS Variables, Glassmorphism, Animations)
- **Font**: Inter (Google Fonts)
- **Icons**: Boxicons, Font Awesome
- **GitHub Stats**: GitHub Readme Stats API, GitHub Calendar

## 🚀 Getting Started

### Prerequisites
- Node.js ≥ 18
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Priyap1038/Priyap1038.github.io.git

# Navigate into the project
cd Priyap1038.github.io

# Install dependencies
npm install

# Start dev server
npm run dev
```

The site will be available at **http://localhost:5173**

### Build for Production

```bash
npm run build
```
